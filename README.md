ОБЩЕЕ ОПИСАНИЕ

1.1 Назначение программы
Программа предназначена для работы с разреженной матрицей в формате SSS (Structurally Symmetric Sparse). Выполняет обход матрицы по варианту 4 и выводит все элементы больше заданного значения b.
1.2 Формат SSS
Структурно-симметричная разреженная матрица, где:
•	Хранится только верхняя треугольная часть + диагональ
•	Экономия памяти до 50% для симметричных матриц
•	При установке a[i,j] автоматически устанавливается a[j,i]
1.3 Вариант обхода 4
Зигзагообразный обход по строкам:
•	Чётные строки (0, 2, 4...): слева направо
•	Нечётные строки (1, 3, 5...): справа налево
